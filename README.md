# my_final_project
# Анализ динамики зарплат и инфляции (итоговый проект)
 
## Описание проекта
 
Проект посвящен анализу изменения средних зарплат в различных секторах экономики и их корректировке с учетом индекса инфляции. Он включает в себя компоненты для визуализации данных и вычислений, которые помогут пользователям лучше понять, как инфляция влияет на реальную покупательную способность населения.
 
## Датасеты
 
Данные о зарплатах взяты с официального сайта Росстата, который содержит статистические данные о средних зарплатах по различным отраслям экономики Российской Федерации. Источник данных: [Росстат](https://rosstat.gov.ru/labor_market_employment_salaries)
 
Данные об инфляции представлены на сайте с информацией об инфляционных показателях в России. Источник данных: [Инфляция в России](https://xn----ctbjnaatncev9av3a8f8b.xn--p1ai/%D1%82%D0%B0%D0%B1%D0%BB%D0%B8%D1%86%D1%8B-%D0%B8%D0%BD%D1%84%D0%BB%D1%8F%D1%86%D0%B8%D0%B8)
 
## Используемые библиотеки
 
- `pandas`: для работы с датасетами.
- `matplotlib`: для визуализации данных.
- `streamlit`: для создания интерактивного веб-приложения.
 
## Компоненты
 
- **Streamlit-приложение**: позволяет пользователям выбирать отрасли и просматривать графики изменений зарплат, как в номинальном, так и в реальном выражении.
- **Jupyter Notebook**: используется для предварительного анализа данных, тестирования и разработки функций визуализации данных.
 
## Функционал
 
- Возможность выбора конкретной категории для анализа зарплат.
- Отображение графика изменения номинальных зарплат по годам.
- Отображение графика изменения реальных зарплат с учетом инфляции по годам.
- Представление анализа в виде текста с использованием Markdown для улучшения читаемости.
